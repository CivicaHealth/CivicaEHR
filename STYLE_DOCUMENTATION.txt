# ClearChart EHR Style Documentation

This document outlines the design system and styling patterns used in the ClearChart EHR application.

## Design Principles
- **Font Family:** DM Sans (sans-serif) for general text, DM Mono (monospace) for numerical values/IDs.
- **Color Palette:** Focused on a professional "Navy" theme with functional accent colors.
- **Layout:** Fixed sidebar with a fluid main content area.

## Color Palette (CSS Variables)
- Primary Navy: `--navy` (#1a2535)
- Navy Mid: `--navy-mid` (#243044)
- Navy Light: `--navy-light` (#2e3d55)
- Slate/Text Secondary: `--slate` (#4a5c72) / `--text-secondary` (#5a6a7e)
- Muted/Border: `--muted` (#8496ab) / `--border` (#dce3ec)
- Background: `--bg` (#f4f6f9)
- Surface (White): `--surface` (#ffffff)
- Accent (Blue): `--accent` (#2f80ed) / `--accent-light` (#e8f0fd)
- Semantic Colors:
  - Success: `--success` (#27ae60) / `--success-bg` (#eaf7ef)
  - Warning: `--warning` (#e67e22) / `--warning-bg` (#fdf3e7)
  - Danger: `--danger` (#e74c3c) / `--danger-bg` (#fdecea)

## Layout Components
- **Sidebar (`.sidebar`):** Fixed width (230px), min-height 100vh, dark background (`--navy`).
- **Main Wrap (`.main-wrap`):** Flex column, margins adjusted for sidebar width.
- **Top Bar (`.topbar`):** Sticky positioning at the top, contains page title and actions.
- **Content Area (`.page-body`):** Padding (28px) for consistent spacing.

## UI Components
- **Cards (`.card`):** White background, subtle shadow (`--shadow`), rounded corners (`--radius`).
- **Buttons (`.btn`):**
  - Primary: Blue background, white text.
  - Secondary: White background, slate text, border.
  - Ghost: Transparent background, blue text.
- **Forms:** Uniform input styling, focus effects (border color + box shadow), textarea resize.
- **Tables (`.data-table`):** Compact, border-collapse, light background for headers (`--bg`).
- **Badges (`.badge`):** Rounded pills with background variants based on semantic colors.
- **Modals (`.mini-modal`):** Fixed overlay, centered card, subtle entry animation.

## Utilities
- **Flexbox & Spacing:** Standardized classes for layout (e.g., `.flex`, `.mt-4`, `.gap-2`).
- **Typography:** Classes for muted text (`.text-muted`), small text (`.text-sm`), and monospace (`.font-mono`).
